Introduction à Git

Git est un système de gestion de versions permettant de suivre l’évolution d’un projet, de collaborer efficacement et d’éviter les pertes de code. Il aide les développeurs à travailler en parallèle, revenir en arrière si nécessaire, et organiser proprement l’historique du projet.

Installation

  1. Télécharger Git depuis le site officiel : https://git-scm.com
  2. Lancer l’installeur et suivre les étapes proposées
  3. Vérifier l’installation avec la commande git --version
