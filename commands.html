Commandes Git essentielles

Liste des commandes :

  • git init
  • git add
  • git commit
  • git status
  • git branch
  • git merge
  • git push

Tableau des commandes

Commande Description
git init Initialise un dépôt Git dans le dossier courant.
git add Ajoute les fichiers à l'index (zone de staging).
git commit Enregistre les modifications dans l'historique.
git status Affiche l'état des fichiers (modifiés, suivis...).
git branch Liste ou crée des branches.
git merge Fusionne une branche dans une autre.
